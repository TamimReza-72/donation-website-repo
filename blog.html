Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTMLdocument as a tree of objects.

How do you select an element from the DOM?

The getElementById method is used to select an element with a specific id.
The getElementsByClassName method returns an array-like collection of elements that match the specified class name.
The getElementsByTagName method returns an array-like collection of all elements with the specified tag name,
The querySelector and querySelectorAll methods are powerful because they allow you to use any valid CSS selector to select elements.
The getElementsByName method returns a collection of elements with a matching name attribute.

What is event delegation in the context of th DOM,and why is it useful?

Event delegation is a technique in JavaScript and DOM manipulation where you add a single event listener to a parent element to manage events triggered by its child elements.

How do you manipulate an element's attributes and styles using the DOM?

By using, setAttribute(attributeName, value),getAttribute(attributeName),removeAttribute(attributeName),style,classList.
